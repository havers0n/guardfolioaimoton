# Нарративная анимация - Experience First Video

Этот проект представляет собой 15-секундную анимацию-нарратив для рекламного видео приложения Guardfolio AI - анализатора рисков инвестиционного портфолио.

## Структура анимации

Анимация разделена на 5 фаз по 5-10 секунд каждая:

### Фаза 1: DISTORTION (0-10 сек)
**Текст:** "Something feels off."
**Визуал:** Искажения, шум, размытие интерфейса
- Интерфейс сильно размыт и искажен
- Наложение шума (noise layer)
- Glitch эффекты
- Интерфейс виден, но нечитаем

### Фаза 2: TENSION (10-15 сек)
**Текст:** "You can't explain it."
**Визуал:** Разрыв, напряжение, максимальные искажения
- Максимальные искажения (размытие, контраст, яркость)
- Эффекты разрыва (красные линии)
- Glitch эффекты с RGB-смещением
- Интерфейс почти невидим

### Фаза 3: PAUSE (15-20 сек)
**Текст:** "But it's there."
**Визуал:** Пауза, тишина, постепенное очищение
- Эффекты постепенно убираются
- Интерфейс начинает проясняться
- Переход к следующей фазе

### Фаза 4: REVELATION (20-25 сек)
**Текст:** "Until you see it."
**Визуал:** Появляется структура UI
- Интерфейс становится видимым и читаемым
- Появляется прогресс-бар и задачи
- Эффекты полностью убраны
- Начинается анимация загрузки

### Фаза 5: CLARITY (25-30 сек)
**Текст:** "See your real risk."
**Визуал:** Полная ясность + Логотип
- Интерфейс полностью чистый и функциональный
- Показывается прогресс анализа
- На 25-й секунде появляется логотип "Guardfolio AI"
- Финальный экран с текстом "Understand your portfolio."

## Архитектура

### Компоненты

1. **NarrativeTimeline** - Управляет таймлайном и переключением фаз
2. **VisualEffects** - Визуальные эффекты (шум, glitch, разрывы)
3. **NoiseLayer** - Canvas-based шум для реалистичности
4. **NarrativeOverlay** - Текстовые наложения с фразами
5. **DistortedInterface** - Применяет эффекты искажения к интерфейсу
6. **InterfaceContent** - Основной контент интерфейса (прогресс, задачи)
7. **LogoFinal** - Финальный экран с логотипом

### Типы и константы

- `NarrativePhase` - Enum фаз анимации
- `NARRATIVE_TIMELINE` - Константы таймлайна (временные метки, сообщения)

## Запуск

```bash
npm install
npm run dev
```

Анимация запускается автоматически при загрузке страницы и длится 15 секунд (DURATION_MS в src/constants.ts).

## Настройка

Таймлайн и сообщения можно изменить в `constants.ts`:

```typescript
export const TIMELINE = {
  DURATION_MS: 15_000,
  FADE_IN_MS: 200,
  FADE_OUT_MS: 200,
  HOOK: { fromMs: 0, toMs: 700 },
  OFF: { fromMs: 700, toMs: 3_000 },
  // ... см. src/constants.ts для полной структуры
}
```

Интенсивность и параметры эффектов настраиваются в компонентах `VisualEffects` и `DistortedInterface`.
